# Telecom Data Enrichment

Python pipeline created to enrich telecom provider databases using ASN matching, WHOIS queries, DNS analysis and ERP inference.

## Features

- ASN enrichment
- PeeringDB integration
- Registro.br WHOIS lookup
- ERP inference via DNS
- Fuzzy matching for provider names
- Contact email extraction
- Logging and caching system

## Technologies

- Python
- Pandas
- Requests
- Socket
- TheFuzz

## Objective

Improve CRM and telecom provider databases by identifying:
- ASN information
- ERP providers
- Technical contacts
- Network-related metadata

## Notes

All sample data used in this repository is fictional and created for portfolio purposes.
